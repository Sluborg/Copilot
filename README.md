# Copilot

Microsoft 365 Copilot actions and extensions.

## Prerequisites

- [Microsoft 365 Agents Toolkit](https://marketplace.visualstudio.com/items?itemName=TeamsDevApp.ms-teams-vscode-extension) (VS Code extension)
- [Teams App CLI](https://www.npmjs.com/package/@microsoft/teamsapp-cli) (`npm install -g @microsoft/teamsapp-cli`)
- Node.js 18+
- Microsoft 365 Developer account

## Getting Started

Use the M365 Agents Toolkit extension in VS Code to scaffold new Copilot actions.
